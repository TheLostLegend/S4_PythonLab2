Лабораторная работа №2
В данной лабораторной работе были написаны сериализаторы json и pickle. Каждый из сериализаторов реализует следующие методы:
    dump, dumps   —   сериализует   Python   объект   в   файл/строку 
    load, loads   —   десериализует   Python   объект   из   файла/строки
    
Сериализация/десериализация: 
    класса, объекта   с   простыми   полями, объекта   со   сложными   полями   и   функциями, функции
    
Юнит-тесты показали coverage 95%.
Консольная утилита умеет конвертировать файлы из одного поддерживаемого формата в другой.

Файлы проэкта:
    newJDecode, newJEncode - сериализатор json
    newPickler - сериализатор pickle
    consoleParser, ParserTests - консольное приложение и его тесты
    json_tests, tests_runner_json - тесты json и файл их запуска
    pickle_tests, tests_runner_pickle - тесты pickle и файл их запуска
    testsMaterial - переменные для проведения тестов
    setup.py - установщик библиотек
    test.pickle, test.json, testBuf.json - примеры работы программы
